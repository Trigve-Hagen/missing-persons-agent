There are three vector databases. investigation_db, investigator_db and determinator_db. You can switch between them to view whats in them. Switching between them will only effect what is shown on this page. If you need to reset any of them for any reason delete the folder in {{ appData }}\databases and the application will rebuild it for you when you restart it. If you delete the {{ appData }}\databases\investigation_db folder you will have to resave all your vectors again. Deleting the vector databases does not affect the data stored in the SqlAlchemy database.
