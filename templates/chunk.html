All Chunks

There is one database investigation_db store in {{ appData }}\databases. There are three collections. Missing Persons holds all the investigation data. Database Statements holds create statements to give the model the information it needs to know to create insert statements for parsed json data that will to be turned into investigation data. Investigation Instructions are instructions you save to train the model on how you want it to run the investigation. The idea is that someone who has experience investigating missing persons might have techneques they regularly use and want the model to use. You can switch between the collections to view whats in them.
